# AOMNIYA
Открытый прототип AGI. Минимальный, воспроизводимый, проверяемый.

## Принцип
Сначала Порядок, затем Сила. Архитектура из четырёх опор:
- **Phi (φ-иерархия):** уровни {L0…Lk} растут по коэффициенту ~1.618 для устойчивости.
- **C (causal order):** вычисления идут по ориентированному ацикличному графу (DAG).
- **Spec (spectral):** входы переводятся в единое спектральное пространство.
- **K (codes & memory):** кодовая маршрутизация, целостность, контент-адресная память.

## Структура
- `src/phi_hierarchy.py` — размеры уровней и их роли  
- `src/causal_dag.py` — граф исполнения и топологический порядок  
- `src/spectral.py` — базовый спектр (rFFT)  
- `src/k_codes.py` — хэш-коды/идентификаторы  
- `src/memory.py` — простая CAM-память  
- `src/agent.py` — склейка Phi → C → Spec → K → результат  
- `docs/` — ROADMAP, SAFETY  
- `tests/` — простые проверки (smoke)
